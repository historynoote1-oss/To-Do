# مجلد اختياري

في الوضع الافتراضي، الأذان في التطبيق بيتشغل عن طريق ستريمنج نفس رابط
القارئ اللي المستخدم مختاره من صفحة "مواقيت الصلاة" (روابط
cdn.aladhan.com) — مفيش داعي تضيف أي ملف هنا.

لو عايز صوت احتياطي يشتغل جوه التطبيق حتى من غير نت، ضيف ملف `.mp3`
باسم `adhan_default.mp3` في المجلد ده، وسكريبت
`frontend/scripts/apply-android-native.mjs` هينسخه تلقائيًا لمشروع
android/ عند أول Build بعد كده.
